LDAP GUI for Academic Institutes

.: Client Installation Steps:.

Follow the same installation procedure on all the machines that you want to configure as the LDAP client. The steps are well-documented and explained in the Report[Section 7.3].
  1. Install the package libnss-ldap by
    sudo apt-get install libnss-ldap
    • Enter the following when installation is going on: ldap://your-ip-address
    • Select Version 3
    • Answer No to all other options

  2. Copy the file named Client.sh and directory named ClientFiles on each machine and then run the Client.sh file.
    ./Client.sh
    The details of the prodecure are provided in the Report[Section 7.3.2]
On successfully running the above shell script, you are done with Client Installation.
